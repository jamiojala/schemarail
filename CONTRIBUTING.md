# Contributing

Thanks for contributing to `schemarail`.

## Local setup

```bash
pnpm install
```

## Commands

```bash
pnpm typecheck
pnpm test
pnpm build
pnpm check
```

## Changesets

This repository uses Changesets for versioning and release notes.

```bash
pnpm changeset
pnpm version-packages
```

## Pull requests

1. Keep changes focused and test-covered.
2. Update docs/examples when behavior changes.
3. Add a changeset for user-facing changes.
